## Analysis Methodology

Each mod was analyzed for archetype compatibility based on:

- **Stat Modifications:** Which stats the mod's HEL equation affects
- **Keywords:** Relevant terms in the mod's name and description
- **Scoring System:** Mods need a minimum score threshold to be associated with an archetype

**Note:** Many mods support multiple archetypes due to cross-archetype synergies.
